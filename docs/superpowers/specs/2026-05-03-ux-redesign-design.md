# GetNote Importer UX 重新设计规范

**日期**: 2026-05-03
**状态**: 已批准

## 概述

对 GetNote Importer 插件的设置界面进行 UX 优化，解决当前界面存在的问题，提供更清晰的状态反馈和用户体验。

## 设计原则

1. **认知简单** — 用户只需要知道"填凭证"，不管背后用什么方式
2. **状态明确** — 实时反馈操作进度，不阻塞用户
3. **信息分层** — 认证、同步配置、操作区、操作结果分层清晰展示

---

## 1. 凭证区设计

### 现状问题
- OAuth 和手动输入是两个"模式"，增加了认知负担
- OAuth 本质是"帮填表单"，不需要单独成为一个模式

### 设计方案
- 认证只需要一个区块
- OAuth 按钮放在凭证输入框旁边，可随时点击授权
- 授权成功/失败的状态要清晰

### 实现
```
┌─────────────────────────────────────────────┐
│ 凭证设置                                     │
├─────────────────────────────────────────────┤
│ [Client ID 输入框] [API Token 输入框]        │
│                                             │
│ [👋 OAuth 授权]  或手动填写上方字段          │
│ ✓ 凭证已配置                                │
└─────────────────────────────────────────────┘
```

---

## 2. 文件名前缀

### 设计方案
- 支持任意字符串，不限于日期格式
- 示例：`GetNote_` + `我的笔记` = `GetNote_我的笔记.md`

### 实现
```
文件名前缀
[ 输入框: GetNote_ ]
💡 前缀会自动加在文件名开头
```

---

## 3. 同步起始日期

### 现状问题
- 用户设置了最大同步天数，但同步时还是会往前翻很久
- 需要指定具体日期来限制同步范围

### 设计方案
- 新增"同步起始日期"配置项
- 只同步此日期之后的笔记
- 留空则同步全部

### 实现
```
同步起始日期
[ ____________ ] 选择日期
💡 只同步此日期之后的笔记，留空表示同步全部
```

---

## 4. 自动同步配置

### 现状问题
- 勾选"自动同步"后，看不到同步周期等配置项

### 设计方案
- 开关打开后，下方展开显示配置项
- 显示：同步周期、启动时同步、上次同步时间

### 实现
```
┌─────────────────────────────────────────────┐
│ 自动同步                      [开关: ON]    │
├─────────────────────────────────────────────┤
│ 同步周期                        每 5 分钟   │
│ 启动时同步                          ✓      │
│ 上次同步                       10分钟前    │
└─────────────────────────────────────────────┘
```

---

## 5. 同步操作与进度条

### 现状问题
- "立即同步"点启动后，弹窗方式不够直观

### 设计方案
- 按钮文案保持"立即同步"（按钮本身已足够清晰）
- 进度条内联显示在按钮下方
- 使用 `#` 字符填充进度条

### 实现
```
[ ⚡ 立即同步 ]

┌─────────────────────────────────────────────┐
│ 正在获取笔记... 第2页              [取消]   │
│ [############░░░░░░░░░] 45%               │
│ 12 / 28 条笔记                            │
└─────────────────────────────────────────────┘

// 完成后
┌─────────────────────────────────────────────┐
│ ✅ 同步完成                                 │
│ 新建 3 · 更新 12 · 跳过 5 · 失败 0         │
│ (5秒后自动收起)                            │
└─────────────────────────────────────────────┘
```

### 进度条设计
- 使用 `#` 字符填充
- 方括号包裹：`[################░░░░]`
- 灰色 `#` 代表已填充，浅色代表空白
- 显示百分比 + 具体数字
- 取消按钮在右上角

---

## 6. 选择性同步

### 设计方案
- 保持现有功能
- 点击按钮打开笔记选择弹窗
- 页面关闭时自动停止后台任务

---

## 配置项顺序

最终顺序：
1. 凭证设置（Client ID + API Token + OAuth 按钮）
2. 目标文件夹
3. 文件名前缀
4. 同步起始日期（新增）
5. 自动同步配置（开关 + 展开配置）
6. 立即同步按钮 + 进度条
7. 选择性同步
